Tip Calculator (Python)

This is a simple Tip Calculator project built using Python.
The program calculates how much each person should pay by splitting the total bill (including tip) among multiple people.

This project was created and tested using Google Colab and is suitable for Python beginners.

Features

-Takes bill amount as input

-Takes number of people sharing the bill

Takes tip amount

=Calculates total bill

=Splits the bill equally per person

How It Works

-User enters the total bill amount

-User enters the number of people

-User enters the tip amount

Program calculates:

-Total bill = bill amount + tip

-Per person cost = total bill รท number of people
